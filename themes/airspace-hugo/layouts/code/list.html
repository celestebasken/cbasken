How I made this website

I made this website independently in the programming language R and some HTML and CSS. Basically, I used my knowledge from my internship with The Nature Conservancy to make a simple website. I've also created a blog documenting my experience in R.
In R, I used the Hugo Blogdown package and Themefisher’s Airspace theme. You can see my Github with all my code here. I don’t profess to be a web development expert, but I’ve also learned a lot about HTML and CSS in the process.
